# Домашнее задание 3

Репозиторий содержит веб-архив сайтов школ, расположенных в районе Теплый Стан города Москва, собранный с помощью библиотеки [wpull](https://github.com/ArchiveTeam/wpull).\
\
## Использовавшиеся ресурсы:\
1. Библиотека wpull
2. Библиотека metawarc
3. Сайт ArchiveReady\
\
## Описание репозитория:\
Каждая папка посвящена одному сайту, название папки является адресом сайта. Внутри каждой папки находятся следующие файлы: [название сайта].cdx, [название сайта].db, [название сайта].txt, metawarc.png. В файле с разрешением txt лежит ссылка на файл архива [название сайта].warc.gz. В файле metawarc.png показываются результаты анализа с помощью библиотеки metawarc
